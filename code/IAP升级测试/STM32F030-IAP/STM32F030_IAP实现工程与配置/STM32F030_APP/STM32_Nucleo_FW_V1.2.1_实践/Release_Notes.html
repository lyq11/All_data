STM32 Nucleo Board Firmware Application Package Update History

  

The STM32 Nucleo Board Firmware Applications Package provides ready-to-run firmware examples to support quick evaluation and development on STM32 Nucleo board. It support many devices, each one contains the following examples:
  • STM32 Nucleo peripheral firmware examples, under Projects\NUCLEO-xxxxx, for more details about each application refer to associated readme file
  • STM32 Nucleo Board Demonstration firmware, under Projects\NUCLEO-xxxxx\Demonstration, for more details about this application and the overall package structure refer to the associated document "Getting started with the STM32 Nucleo board firmware package (UM1726)"
For more information on the STM32 Nucleo, to download the available firmware package and documentation visit www.st.com/stm32nucleo.

1.2.1 / 20-June-2014

 

Main Changes

• Fix driver : map LED2 to PB13 pin for NUCLEO-F302R8 board

   • Important Note: ST-LINK/V2-1 integrated in STM32 Nucleo Board is not yet supported by  MDK-ARM v4.73.0 used for this package development,

     it will be supported from MDK-ARM v5.10.  Currently, user must install ST-LINK/V2-1 driver available on www.st.com to be able to use STM32 Nucleo board.

Content

• CortexM CMSIS V3.20 (Release Notes)

• STM32F0xx Standard Peripherals Driver V1.3.0 (Release Notes)

• CMSIS_Device_STM32F0xx V1.3.2 (release notes)

• STM32F30x Standard Peripherals Driver V1.1.1 (Release Notes)

• CMSIS_Device_STM32F30x V1.1.1 (release notes)

• STM32F10x Standard Peripherals Driver V3.6.1 (Release Notes)

• CMSIS_Device_STM32F10x V3.6.3 (release notes)

• STM32L1xx Standard Peripherals Driver V1.3.0 (Release Notes) 

• CMSIS_Device_STM32L1xx V1.3.2 (release notes)

• Demonstration & Peripheral firmware examples firmware V1.2.0 (release notes)

• STM32 Nucleo Drivers V1.1.1 (release notes)

 

Development Toolchains and Compilers

• Atollic TrueSTUDIO STM32 (TrueSTUDIO) toolchain v4.1.0

• IAR Embedded Workbench for ARM (EWARM) toolchain v7.10.1

• Software patches are needed to use STM32L152RE and STM32F302R8 devices with MDK-ARM v4.73.0. If you do not have

these patches installed, you can install them from the following folder :

"\Utilities\Third_Party\MDK-ARM_STM32L152RE_Patch for STM32L152RE device

"\Utilities\Third_Party\MDK-ARM_STM32F302R8_Patch for STM32F302R8 device

• RealView Microcontroller Development Kit (MDK-ARM) toolchain v4.73.0

 

Supported Devices and EVAL boards

•  STM32nucleo board Rev C. 

•  Devices supported : STM32F302R8, STM32FO30R8, STM32F103RB and STM32L152RE.

 

Known Limitations

•   None.
